# Contributing to NEXO

First off, thank you for considering contributing to NEXO! It's people like you that make NEXO such a great tool.

## Where do I go from here?
If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/ANUPAM4545/Industrial--Knowledge-AI/issues) first.

## Fork & create a branch
1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Commit your changes (`git commit -m 'Add amazing feature'`)
5. Push to the branch (`git push origin feature/amazing-feature`)
6. Open a Pull Request
